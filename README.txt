New in version 0.3: 
* Collision functionality between bottom of pit / other inactive blocks and active block added.
* Textures added.
* Some minor code improvements.


How to run (visual studio): 
	1. Copy repository web link
	2. Open Visual Studio
	3. Select "Clone a repository"
	4. Add repository link to "repository location" and click "clone"
	5. When is finished CMake configuring the project, select "exam.exe" as
	   startup item, then run it
	6. Now, the program should run


Controls:
ESC: Exits game
W: Up
S: Down
A: Left
D: Right
Space: Move block
